windows设置免密码登陆linux

windows

  1. 安装git（gitbash可以试用ssh，scp等）
  2. ssh-keygen -t rsa -C '邮箱' ，然后三下enter键
  3. 在当前用户文件夹下地./ssh文件中找到id_rsa.pub

linux

  1. 检查用户文件夹（root）的.ssh有没有authorized_keys，没有话手动创建
  2. vi authorized_keys，打开文件复制，wq退出
  3. 重启ssh服务service sshd restart
