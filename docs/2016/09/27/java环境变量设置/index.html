java环境变量设置

工作需要本地运行java开发

  • 第一步：
    右击【我的电脑】—【属性】—–【高级】—【环境变量】

  • 第二步：
    【新建系统变量】，在“变量名”文本框输入“JAVA_HOME”,在“变量值”文本框输入JDK的安装路径

  • 第三步：
    设置Path变量 最前边输入”%JAVA_HOME%in;“

  • 第四步：
    在“系统变量”选项区域中查看CLASSPATH 变量，如果不存在，则新建变量CLASSPATH，否则选中该变量，单击“编辑”按钮，在“变量值”文本框的起始位置添加“.;%JAVA_HOME%libdt.jar;%JAVA_HOME%lib ools.jar;”
