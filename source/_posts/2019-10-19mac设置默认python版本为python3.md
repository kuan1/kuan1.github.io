### 一、[下载python3](https://www.python.org/downloads/)

### 二、mac设置python版本为python3
```bash
# 检查命令位置
which python3

# 修改bash_profile文件
vi ～/.bash_profile

# 底部加入alias
alias="【python3命令位置】"

# 重启Termimal
```
